To run:
- Run Ganache
- Run 'npm run start' in one terminal
- Tab will open with EthSwap exchange

Notes:
- 'truffle migrate --reset' to reset the blockchain
- 'truffle compile' to compile
- 'truffle console' to open truffle console and run commands
- 'truffle test' to run test files
- Ganache allows local fake blockchain to run
- MetaMask used as browser extension to allow browser to interact with the blockchain
- For this project use Ganache custom network in MetaMask (i.e. to use fake ETH)
